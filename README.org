* creat-ts-app
Simple script to setup empty ts project.
** Usage
*** In new directory
#+BEGIN_SRC shell
$ creat-ts-app new-dir
#+END_SRC
*** In current directory
#+BEGIN_SRC shell
$ creat-ts-app
#+END_SRC
** What will it do?
It will create a directory (if provided)
It will run these 2 commands then.
#+BEGIN_SRC shell
$ npm i typescript --save-dev
$ npx tsc --init
#+END_SRC
